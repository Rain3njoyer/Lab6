# Contributing to Labs

This is a college assignment project for DevOps labs.

## How to Contribute

### Reporting Bugs
- Use the bug report template when creating an issue
- Include clear steps to reproduce the bug
- Add screenshots if applicable

### Suggesting Enhancements
- Open an issue with the "enhancement" label
- Clearly describe the proposed feature

### Development Setup
1. Clone the repository
2. Ensure Docker is installed
3. Run `docker-compose up` to start the database
4. Follow lab instructions for specific tasks

## Code Style
- Follow Java naming conventions
- Write clear commit messages
- Keep methods focused and well-documented

## Testing
- Run unit tests: `mvn test`
- Run integration tests before committing
- Ensure all tests pass

## Questions?
Open an issue with the "question" label.
