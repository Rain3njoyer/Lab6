# Security Policy

## Reporting a Vulnerability

This is an educational project for college labs. If you discover a security vulnerability:

1. Open an issue describing the vulnerability
2. Do not publicly disclose details until addressed
3. Allow reasonable time for a fix

## Supported Versions

Currently, only the latest version in the `master` branch is maintained.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |
